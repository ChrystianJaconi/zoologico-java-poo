# Zoológico em Java

Projeto desenvolvido por mim durante o curso de Programação Orientada a Objetos (POO) do Bradesco.

## Funcionalidades
- Listagem de animais
- Alimentação
- Emissão de sons
- Exibição de idade
- Movimentação dos animais

## Conceitos aplicados
- Herança
- Polimorfismo
- Encapsulamento
